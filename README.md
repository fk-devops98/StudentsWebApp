## List of students

Interface where the user can see all students in the form of tables with basic information such as:

-   **Username**
-   **FirstName**
-   **LastName**
-   **Age**
-   **Career**

## Student form

Modal where the user can create or edit a student, the required fields are:

-   **Username**
-   **FirstName**
-   **LastName**
-   **Age**
-   **Career**
